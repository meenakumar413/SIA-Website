ChatGPT

Purpose: ChatGPT is an AI language model developed by OpenAI that generates human-like responses to text inputs.

Key Features: Conversation simulation, writing assistance, brainstorming, and learning support.

How It’s Used: Educators and students use ChatGPT to summarize lessons, explain topics, and practice writing or coding tasks.

Notebook LM

Purpose: Notebook LM is Google’s AI-powered research and note-taking assistant.

Key Features: Summarizes long documents, answers questions from your notes, and creates study guides automatically.

How It’s Used: Ideal for students and teachers managing projects or analyzing research materials.

Google Gemini

Purpose: Gemini is Google’s generative AI model designed for reasoning, problem-solving, and multimodal understanding.

Key Features: Works across text, images, and code. Integrates deeply with Google Workspace tools like Docs and Sheets.

How It’s Used: Helps with lesson planning, automating reports, and creative classroom projects.

Microsoft Copilot

Purpose: Microsoft Copilot integrates AI directly into Office tools to boost productivity and creativity.

Key Features: Generates emails, summarizes documents, creates presentations, and analyzes Excel data.

How It’s Used: Perfect for students and professionals managing time, data, and digital communication efficiently.

Perplexity AI

Purpose: Perplexity AI is a conversational search engine that provides clear, sourced answers to questions.

Key Features: Real-time web search, citation links, topic exploration, and research summaries.

How It’s Used: Great for inquiry-based learning and research, ensuring factual and reliable responses.

Git & GitHub

Purpose: Git tracks changes in code (version control), while GitHub hosts projects for collaboration.

Key Features: Save versions, collaborate on repositories, and share code easily online.

How It’s Used: Enables teamwork on coding projects, version history tracking, and portfolio development for students.
